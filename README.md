# Dino Jump Game

## Overview
Dino Jump is a simple 2D game where the player controls a dinosaur and must jump over incoming obstacles (tridents) to survive. The game tests the playerâ€™s timing and reflexes, providing a straightforward yet engaging experience.

## Gameplay
- The player character automatically moves forward.
- Obstacles appear from the right side of the screen.
- **Jump controls:**  
  - Press **Spacebar** or **Click with the Mouse** to jump.
- Colliding with a trident ends the game.

## Features
- Simple and intuitive controls
- Endless runner-style gameplay
- Collision detection for obstacles
- Lightweight and easy to run

## How to Run

```
git clone https://github.com/isa0-gh/lua-dino-game
cd lua-dino-game
love .
```
